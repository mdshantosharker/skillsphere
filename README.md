## ⚙️ Tech Stack

- ⚡ Next.js (App Router)
- 🎨 Tailwind CSS
- 🧩 HeroUI / DaisyUI
- 🔐 BetterAuth
- ✨ Animate.css
- 🔔 Toast Notifications

---

## ✨ Features

- 🔐 Authentication System (Login / Register / Google Login)
- 🔒 Protected Routes (Course Details access control)
- 📚 Course Browsing System
- 🔍 Search Courses by Title
- 👤 User Profile View & Update
- ⚡ Loading states for better UX
- ❌ Custom 404 Not Found Page
- 🎨 Smooth UI animations (Animate.css)
- 📱 Fully responsive design
